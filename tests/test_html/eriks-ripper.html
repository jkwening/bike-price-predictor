DESCRIPTION & FEATURES

Shop all SE Bikes Products

The smallest SE bike ever! The 2018 SE Bikes Micro Ripper Run Bike lets the tiniest of riders get their first feel of the balance of riding a bike. This micro-sized bike is built with legendary SE features such as Floval shaped frame tubing and Landing Gear forks and yet has an ultra-low standover and kid-friendly geometry. The Micro Ripper is a perfect introduction to the SE lifestyle.

  • Lightweight 6061 aluminum frame with ultra-low standover height to correctly fit young riders
  • 12 inch wheels for easy handling and stable steering
  • Adjustable seat height for growing kids
  • 9.4 lbs


Frame6061 Aluminum Floval Tubing, Easy Standover
ForkMicro Landing Gear
Wheels16H Alloy Hubs, Alloy Rims
Tires12" x 2.125" Front & Rear
Headset1-1/8" Threadless
StemTop-Load Alloy, 40mm Reach
Handlebars16" x 2" Alloy
GripsMini Kraton
SaddleSE Racing Mini Seat / Post Combo
Seat PostIntegrated Alloy, 22.2mm
Seat BinderAlloy Quick Release, 31.8mm Inner Diameter
